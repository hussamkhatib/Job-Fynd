## Installation

Clone the project

```bash
git clone https://github.com/hussamkhatib/tap.git
```

Go to the project directory

```bash
cd tap
```

Install dependencies

```bash
yarn
```

Run the frontend 
```bash
yarn dev
```

Run storybook
```bash
yarn storybook
```
