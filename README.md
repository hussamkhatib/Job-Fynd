# TAP portal VVCE 
A Web app which lets students of VVCE create their portfolio using notion as CMS.
## Installation  

Clone the project
```bash
git clone https://github.com/hussamkhatib/tap.git   
```
Go to the project directory
```bash
cd tap
```
Install dependencies
```bash
npm install
```
Start the server
```bash
npm run dev
```
    
## Tech Stack

- Next.js
- typescript
- auth0
- Notion API 
- react-notion
- TailwindCSS
